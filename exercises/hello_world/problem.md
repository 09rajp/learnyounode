## ヒント

Node.jsのアプリのためは新しい`.js`ファイルだけが必要です。書いた後は`node`コマンドを使って動します。例えば：

```sh
$ node program.js
```

このファイルにはブラウザと同じようにJavaScriptが書けます。

```js
console.log("text")
```

アプリができたら以下のコマンドを実行してください：

```sh
$ {appname} verify program.js
```

これを実行するとテストが動きます。プログラムが正しく書けていればこのステップが`completed`になります。
